# Website UMKM
By Marcell, Arielle

PANDORA’S BOX

Selamat datang di Pandora’s Box, sebuah situs yang bertujuan untuk mendukung dan mempromosikan UMKM (Usaha Mikro, Kecil dan Menengah) di sekitar kita. Melalui website ini, kami berharap bahwa masyarakat dapat menggunakanya untuk menemukan produk, toko atau tempat nongkrong baru dengan mudah.

Tech stack:
HTML
CSS
Javascript
Bootstrap

Progress saat ini: Sudah ada landing page dan home page dengan search bar

Cara membuka website: download folder UMKM,buka UMKM.html di browser terpilih
